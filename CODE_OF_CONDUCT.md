# Code of Conduct

## Our standard

PadPulse is an open project maintained with respect, patience and collaboration in mind. We expect everyone participating in the project to:

- Be respectful and constructive.
- Assume good intent and communicate clearly.
- Welcome beginners and help people learn.
- Focus criticism on ideas, code and behavior, not on people.

## Unacceptable behavior

- Harassment, hate speech or personal attacks.
- Doxxing or sharing private information.
- Trolling, intimidation or repeated hostile behavior.
- Deliberately disruptive or bad-faith contributions.

## Enforcement

Project maintainers may remove, edit or reject comments, commits, issues or pull requests that violate this code of conduct.

## Scope

This code of conduct applies to issues, pull requests, discussions and other project spaces.

## Reporting

If you see unacceptable behavior, report it privately to the maintainers through GitHub security reporting or a private maintainer contact channel when available.
